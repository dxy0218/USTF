# Campaign Operations Checklist

**Template Type:** Runtime Checklist  
**Version:** v3.0.0-draft  
**Use With:** `core/15-campaign-operations.md`

## Session Start

```text
[ ] Campaign identified
[ ] Current in-world date identified
[ ] Current location / theater identified
[ ] Active player Actor identified
[ ] Relevant non-player Actors identified
[ ] Current State loaded
[ ] Knowledge Ledger checked
[ ] Open Threads checked
[ ] Relevant Module identified
[ ] Immediate pending decisions identified
```
