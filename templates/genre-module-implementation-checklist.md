## Core Boundary Check

```text
[ ] Module does not override Core Invariants.
[ ] Module does not bypass Knowledge State.
[ ] Module does not bypass Time Continuity.

[ ] Module does not redefine Actor Model incompatibly.
[ ] Module declares all special resources.
[ ] Module declares all special procedures.
[ ] Module declares compatibility warnings.
```

## Campaign Fit

```text
[ ] The module matches the campaign era or genre.
[ ] The module matches the primary mode of play.
[ ] The module does not add unnecessary complexity.
[ ] The module adds useful constraints or procedures.
[ ] The module interacts clearly with current campaign state.
```
