## Public Rulebook Target

The public rulebook should be generated from source files, then lightly edited for readability.

Target length:

```text
50–100 pages
```

Recommended editions:

```text
USTF Core Rulebook
USTF Campaign Operator Guide
USTF Module Developer Guide
USTF Runtime Guide for LLMs
```

## Editorial Rule

Public prose may explain and illustrate.

Public prose must not contradict Core Specification.

If explanatory text conflicts with numbered Core rules, Core rules prevail.

---

## Example Placement

Examples should be clearly marked:

```text
Example — Non-binding
```

Playtest examples should be separated from generic rules.

Private campaign material should not be copied directly into public release without cleanup.
