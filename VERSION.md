## Version Meaning

USTF uses a semantic versioning-inspired policy.

```text
MAJOR.MINOR.PATCH
```

- MAJOR changes may break Core compatibility.
- MINOR changes add backward-compatible rules, modules, or procedures.
- PATCH changes clarify, correct, or tighten existing rules without changing meaning.
