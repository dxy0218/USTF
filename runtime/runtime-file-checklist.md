# USTF Runtime File Checklist

Use this checklist before starting or continuing a campaign inside ChatGPT Project.

---

## 1. Required Runtime Files

```text
[ ] Runtime pack loaded
[ ] Current campaign state loaded
[ ] Knowledge ledger loaded
[ ] Open threads loaded
[ ] Relevant module loaded
```

---

## 2. Recommended Runtime Files

```text
[ ] Actor registry loaded
[ ] Organization state loaded
[ ] Recent session delta loaded
[ ] Session delta template loaded
[ ] Selected reference procedure loaded
```

---

## 3. Before Play

```text
[ ] Confirm in-world date/time
[ ] Confirm active player Actor
[ ] Confirm immediate scene or strategic state
[ ] Confirm unresolved open threads
[ ] Confirm known constraints
[ ] Confirm relevant non-player Actors
```

---

## 4. During Play

```text
[ ] Check Actor knowledge before action
[ ] Check resources before major commitment
[ ] Check time scale
[ ] Check opposition
[ ] Record consequences
[ ] Advance relevant non-player Actors
[ ] Preserve Unknown when unresolved
```

---

## 5. End of Session

```text
[ ] Generate Session Delta
[ ] Update Current State
[ ] Update Knowledge Ledger
[ ] Update Open Threads
[ ] Update Actor/Organization state if needed
[ ] Mark continuity issues
[ ] Archive raw events only after summary
```
