# USTF Changelog

## v3.0.0-draft

### Added

- Established USTF as a campaign engine specification rather than a traditional RPG rulebook.
- Added source-first, LLM-executable, book-ready delivery model.
- Added ChatGPT Project runtime instructions.
- Added runtime quickstart procedure.
- Added minimal core pack.
- Added core invariants.
- Added architecture principles.
- Added public API concept.
- Added session delta template.
- Added knowledge ledger template.
- Added campaign state pack template.
- Added first ADRs for Actor Model and LLM Deployment Layer.


### Licensing

- Selected CC BY 4.0 for documentation, rules, templates, modules, ADRs, and book material.
- Selected MIT License for code, scripts, build tooling, and automation.
- Replaced Decision Pending license placeholder with explicit split-license policy.

### Design Notes

USTF v3.0.0-draft is not yet a complete public rulebook.

It is the first stable architecture baseline for building the full system.
